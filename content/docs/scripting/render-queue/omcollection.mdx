---
title: "OMCollection object"
---

`app.project.renderQueue.items.outputModules`

#### Description

The OMCollection contains all of the output modules in a render queue. The collection provides access to the [OutputModule objects](outputmodule.mdx), and allows you to create them. The first OutputModule object in the collection is at index position 1.

<Callout>OMCollection is a subclass of [Collection object](../other/collection.mdx). All methods and attributes of Collection are available when working with OMCollection.</Callout>

---

## Methods

### OMCollection.add()

`app.project.renderQueue.item(1).outputModules.add()`

#### Description

Adds a new Output Module to the Render Queue Item, creating an OutputModule.

#### Returns

[OutputModule object](outputmodule.mdx).
